# DNS Traffic Analysis

This project demonstrates DNS traffic analysis using Wireshark.

## What I did:
- Captured DNS traffic
- Analyzed DNS queries and responses
- Observed how domain names are resolved into IP addresses

## Example:
- Domain: neverssl.com
- Protocol: DNS
- Port: 53

## Tools:
- Wireshark

This is my first cybersecurity project focused on network analysis.
